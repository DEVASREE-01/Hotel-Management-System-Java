# Hotel Management System - Java OOP Project

## Description
This is a **console-based Hotel Management System** developed using **Core Java**.  
The project simplifies hotel operations by allowing users to manage rooms, bookings, food orders, and billing efficiently. It demonstrates strong **OOP concepts**, exception handling, and menu-driven interface design.

---

## Features

- **Room Management**  
  - Check room availability  
  - Book single and double rooms (Luxury & Deluxe)

- **Customer Management**  
  - Store customer details (name, contact, gender)  
  - Multiple customers in double rooms

- **Food Ordering**  
  - Order items from the menu (Sandwich, Pasta, Noodles, Coke)  
  - Calculates food charges automatically

- **Billing & Checkout**  
  - Automated billing combining room charges and food orders  
  - Checkout and deallocate rooms

- **Data Persistence**  
  - Saves customer and booking data using **file handling**  
  - Ensures data is retained across program runs

---

## Technologies Used

- **Java (Core Java)**  
- **Object-Oriented Programming (OOP)**  
- **File Handling for data storage**  
- **Exception Handling & Input Validation**  
- **Arrays for room & customer management**
